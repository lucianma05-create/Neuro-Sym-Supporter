# Neuro-Sym Supporter
This is the repository of WWW'2026 submission paper #rfp1651: "Neuro-Sym Supporter: A Thoughtful Emotion Support Agent Integrating Neural  and Symbolic Policy Learning"

## Running :

\#  for nero_mind
```yaml
python train_nero_mind.py
```
\#  for sym_mind
```yaml
python train_sym_mind.py
```
\#  for blend
```yaml
python train_blend.py
```

\#  training model
```yaml
cd LlaMA-Factory
python src/train.py
```
